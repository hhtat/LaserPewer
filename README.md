# LaserPewer

Easy to use software for GRBL based laser cutters.

## Features

* Grbl v1.1
* SVG import
* Vector cutting
* Path optimizer

### Planned

In priority order:

* Installer (NSIS)
* Raster engraving
* Multiple configurations (color-based filters)
* Z-axis support

Wish list:

* OS X and Linux support (Mono or Electron)
